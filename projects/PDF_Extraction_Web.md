# PDF_Extraction_Web

> Public repository note

## What this repo is about
PDF 추출과 웹 기반 문서 처리 흐름을 실험한 공개 저장소입니다.

## Why it matters
문서형 데이터는 RAG나 검색 시스템에서 자주 등장하기 때문에,  
이 저장소는 문서 추출 / 가공 / 웹 처리 흐름에 대한 기본 실험을 보여줍니다.

## What to look for
- PDF 추출 흐름
- 문서 처리와 웹 인터페이스 연결
- 후속 검색 / 요약 / 가공 파이프라인의 전처리 관점
