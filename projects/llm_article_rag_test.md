# llm_article_rag_test

> Public repository note

## What this repo is about
이 저장소는 RAG 구조 실험, 검색 품질 검토, 재정렬 아이디어를 테스트하기 위한 공개 실험 저장소입니다.

## Why it matters
실제 회사 프로젝트는 비공개인 경우가 많기 때문에, 이 저장소는 다음과 같은 관심사를 공개적으로 보여주는 역할을 합니다.

- RAG 파이프라인 구조에 대한 관심
- retrieval 품질을 단순 생성보다 먼저 보는 관점
- 재정렬(rerank)과 검색 품질 실험
- 실제 운영형 프로젝트에 적용되기 전의 실험 아이디어

## What to look for
- 검색 구조 실험
- retrieval 품질 검토
- rerank 아이디어
- 파이프라인 비교 관점
