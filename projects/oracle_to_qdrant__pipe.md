# oracle_to_qdrant__pipe

> Public repository note

## What this repo is about
Oracle 데이터 전처리, 임베딩 생성, Vector DB 적재 흐름을 공개적으로 실험한 저장소입니다.

## Why it matters
실제 운영 환경의 적재 파이프라인은 비공개인 경우가 많지만,  
이 저장소는 벡터 검색 시스템의 앞단 인프라에 대한 관심과 구현 방향을 보여줍니다.

## What to look for
- Oracle 기반 데이터 추출
- 임베딩 생성 흐름
- Vector DB 적재
- 검색 시스템의 데이터 준비 과정
