# Assets

이 폴더는 GitHub 포트폴리오에 들어갈 이미지 자산을 위한 위치입니다.

## 권장 파일
- `profile.png` — 프로필 이미지
- `ntis-rag-architecture.png` — NTIS AI 챗봇 아키텍처 다이어그램
- `everyones-rnd-architecture.png` — 모두의 R&D 파이프라인 다이어그램
- `pipeline-overview.png` — 데이터 적재 / 검색 / 서빙 전체 구조
- `benchmarks.png` — 공개 가능한 메트릭 또는 표 스냅샷

## 사용 팁
- 민감한 서버 정보(IP, 계정, 토큰, 내부 식별자)는 가린 뒤 업로드합니다.
- Mermaid 다이어그램을 PNG로 내보내 README에 함께 첨부하면 가독성이 좋아집니다.
- 같은 이미지를 여러 문서에서 재사용할 경우 상대경로를 통일합니다.
